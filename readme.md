# LLM Tutorial

This repository is a structured tutorial for studying large language models from first principles.

The goal is to build enough background to reproduce a state-of-the-art, completely open-source model, including:

- deep learning fundamentals
- auto-differentiation and tensor calculus
- large language models
- reinforcement learning
- multimodal and physical models
- large-scale training infrastructure
- data engineering
- agentic language models
- core computer science background
- advanced topics such as long context, efficient transformers, reasoning, verification, and self-improvement

## Where To Start

Please go to https://powergiant.github.io/llm_tutorial/#/ for better reading experience.
